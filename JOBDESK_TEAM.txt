ATURAN WAJIB SEBELUM MULAI (PENTING!)

1. Ambil update terbaru dari branch utama (development/main).

   git pull origin development

2. Buat branch sendiri sesuai nama dan tugas.
   🔥 WAJIB: GUNAKAN BRANCH YANG BERBEDA UNTUK SETIAP TUGAS BARU! JANGAN PAKAI BRANCH LAMA! 🔥

   git checkout -b fitur/nama-tugas-nama

3. Jangan push langsung ke main.

4. Setelah selesai, commit dan push branch masing-masing.

   git add .
   git commit -m "menambahkan tugas nama"
   git push origin fitur/nama-tugas-nama

5. Setelah push, buat Pull Request atau minta ketua project merge ke development.
